#  GenAI Domain Assistant Chatbot

An AI-powered chatbot built using OpenAI/Gemini API that supports multi-turn conversations and domain-specific assistance such as HR support and customer service.


##  Project Overview

This project demonstrates how to build a conversational AI chatbot using Generative AI APIs.  
The chatbot can maintain conversation history, answer user queries, and simulate real-world assistants using system prompts.

Two assistants were implemented:

- HR Assistant
- TechShop Customer Support Bot

The project focuses on prompt engineering, chatbot development, and API integration.


##  Features

- AI-powered chatbot
- Multi-turn conversation support
- Conversation history management
- System prompt customization
- HR assistant chatbot
- Customer support chatbot
- Error handling implementation
- Real-time user interaction



##  Technologies Used

- Python
- OpenAI API
- Gemini API
- google-genai SDK
- dotenv
- Jupyter Notebook / Kaggle
